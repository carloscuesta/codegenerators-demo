## Quick start

```bash
git clone https://github.com/carloscuesta/codegenerators-demo.git && npm install
```

### List generators

```bash
npm run generators
```

### Run a generator

```bash
npm run generators NAME ACTION
```

Replace `NAME` and `ACTION` parameters with the generator name and the action that you want to run.
